# ドキュメント用スクリーンショット

## 目的

このフォルダーには、マニュアルや説明用に使用する TeamOps のスクリーンショットを保存します。

## 推奨ファイル

- `dashboard.png`
- `reports.png`
- `production-monitor.png`
- `admin-production.png`
- `admin-panel-machines.png`
- `admin-panel-modal.png`
- `admin-panel-system-log.png`
- `production-monitor-probe.png`

## 注意

スクリーンショットは TeamOps にログインした状態で取得します。HTML 画面は WinForms/WebView2 からデータを受け取るため、単独で開くと実データが表示されない場合があります。

`admin-panel-*` の画像は、ドキュメント用のシミュレーションデータで取得されたものです。運用 DB へ書き込みません。
