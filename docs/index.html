Overview

TeamOps centraliza rotinas de operadores, liderancas e administradores em ambiente industrial. O app principal abre com login, carrega um dashboard HTML dentro do WinForms e direciona para modulos operacionais.

C# .NET 9 WinForms WebView2 Dapper SQLite HTML/CSS/JS
TeamOps.UIAplicacao principal, dashboard, forms e modulos HTML.
TeamOps.OperatorAppApp separado para leitura de Hikitsugui por operador.
TeamOps.CoreEntidades, validadores e tipos comuns.
TeamOps.DataRepositorios, Dapper, SQLite, SQL e migrations.
TeamOps.ConfigCaminhos e configuracao do banco.

Arquitetura

A UI chama Core/Data/Config. Os modulos HTML rodam em WebView2 e se comunicam com C# por window.chrome.webview.postMessage e PostWebMessageAsJson.

Fluxo tecnico

  • O app resolve DatabasePath pelo App.config.
  • DbInitializer.EnsureCreated() aplica schema e garante o banco.
  • O login identifica usuario, operador e turno.
  • O dashboard abre modulos conforme AccessLevel.
  • Repositorios acessam SQLite por SqliteConnectionFactory.
Quando uma regra nao estava explicita no codigo, a documentacao Markdown marca o trecho como comportamento inferido.

Manual do usuario

O uso normal comeca pelo login, passa pelo dashboard e segue para o modulo de trabalho. Os botoes disponiveis dependem da permissao do usuario.

Operacao diariaAbrir sistema, conferir usuario/turno, acessar modulo e salvar registros.
RelatoriosUsar filtros antes de imprimir, exportar ou consultar grandes volumes.
Boas praticasNao mover templates, anexos, exportacoes ou banco sem suporte tecnico.

Troubleshooting essencial

Sistema nao abreVerificar rede, banco, permissoes, antivirus e deploy completo.
WebView brancoReparar WebView2 Runtime e conferir arquivos ui no output.
Erro SQLiteChecar locks, permissao, conectividade e integridade do DB.
Monitor nao atualizaValidar exportacao, pasta, cache do browser e data/hora do HTML.
Importacao falhaConferir BAT, TXT/DAT, encoding, nomes e arquivo import.done.
PR/CL falhaValidar templates, pastas de saida e arquivos Excel bloqueados.

Deployment

O deploy esperado e self-contained, com WebView2 como pre-requisito de estacao. A pasta publicada deve manter DLLs, assets, App.config e todos os subdiretorios ui.

Pastas padrao

  • C:\TeamOps\DB\teamops.db
